League of Legends Degrees of Separation

Python

Created a webapp using Streamlit which allows you to input two summoner names and find out how many degrees of separation they have.

For example, if you played in the same game as a target person, you have one degree of separation. If you played in the same game as someone else who played in the same game as your target person, you have two degrees of separation with your target summoner.

Due to Riot API's rate limitations, however, only small degrees of separation can be calculated.
